# Web 版性能分析器 - 执行计划

创建时间：2026-04-18 14:55

## 项目目标

创建基于 Web 的 Linux 性能分析平台，支持：
1. 网页界面配置目标设备
2. 远程执行命令采集数据
3. 调用大模型 API 分析性能数据
4. 支持切换不同大模型（通义千问/DeepSeek/ChatGLM 等）
5. 实时查看分析进度和报告

## 技术栈

- **后端**: Flask + Python
- **前端**: HTML + Vue.js 3 + Element Plus
- **大模型**: 支持多个 API 提供商
- **任务队列**: 异步执行分析任务

## 启动说明

1. 安装依赖：`pip install -r requirements.txt`
2. 配置 API Key：编辑 `config/config.yaml`
3. 启动服务：`cd backend && python app.py`
4. 访问界面：http://localhost:5000
