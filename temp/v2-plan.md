# V2 版本开发计划

## 创建时间
2026-04-25 14:21

## 目标
在 V1 基础上增加更多自定义和扩展能力

## 用户需求功能

### 1. 支持多种自定义模型
- 用户可配置多个 LLM 模型
- 支持动态切换模型
- 支持模型优先级设置

### 2. 支持采集次数、采集间隔设置
- 可配置采集频率
- 可配置总采集次数
- 支持定时采集任务

### 3. 采集命令管理页面
- 显示当前所有采集命令
- 复选框控制开启/关闭
- 可视化配置界面

### 4. 支持自定义采集命令
- 用户可添加自定义命令
- 命令模板库
- 命令验证和测试

## 补充建议

### 5. 数据可视化增强
- 实时性能趋势图
- 历史数据对比
- 异常点自动标记

### 6. 告警系统
- 阈值配置
- 多渠道通知（邮件、钉钉、企业微信）
- 告警历史记录

### 7. 报告导出
- PDF/Excel 格式导出
- 自定义报告模板
- 定时自动生成报告

### 8. 多主机管理
- 主机分组
- 批量操作
- 主机健康状态总览

### 9. 插件系统
- 采集插件接口
- 分析插件接口
- 第三方插件市场

### 10. API 开放
- RESTful API
- Webhook 支持
- 第三方集成

## 技术建议

### 前端
- 使用 Vue3/React 重构
- 增加 WebSocket 实时推送
- 响应式设计支持移动端

### 后端
- 数据库持久化（SQLite/PostgreSQL）
- 任务队列（Celery/Redis）
- 缓存层（Redis）

### 部署
- Docker 容器化
- Docker Compose 一键部署
- K8s 支持
