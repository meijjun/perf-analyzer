## 已知问题修复

### 问题 1: Vue 模板未渲染
**症状**: 页面显示 `{{ target.name }}` 而不是实际值  
**原因**: 对话框 HTML 在 `</script>` 之后  
**修复**: 移动到 `</script>` 之前 ✅

### 问题 2: 默认命令无法删除
**症状**: 点击删除按钮无反应或提示"系统命令不可删除"  
**原因**: 缺少 `is_custom` 字段  
**修复**: 所有命令添加 `is_custom: False` ✅

### 问题 3: 按钮无法点击
**症状**: 按钮看起来正常但点击无反应  
**原因**: CSS z-index 问题  
**修复**: 添加 z-index 确保按钮在最上层 ✅
